## Overview of the project
Write a python program to take a url and example text file and then via a scrape using implied css-identifiers output
a json file of the data and analysis of the data.

## What is the problem?
Given web page containing structured data. Analyse the page to extract css-identifiers for the data.
Then output a json file of the css-identifiers and example data. Paying special attention 
to identifying section of page containing the results.

## What is the solution?

Write AI script with examples to identify css-identifiers for the data. 
Then output a json file of the css-identifiers and example data. 
Paying special attention to identifying section of page containing the results.

e.g. for google search results

from requests_html import HTMLSession
import requests

def get_request(url):
    headers = {
        'Connection': 'keep-alive',
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/79.0.3945.88 Safari/537.36',

        'Accept-Language': 'en-US,en;q=0.9',
        'Accept-Encoding': 'gzip, deflate',
    }
    web_response = None  # Initialize web_response to None
    try:
        session = HTMLSession()
        web_response = session.get(url, headers=headers)
        session.close()
        # response = requests.get(url, headers=headers)

    except requests.exceptions.RequestException as e:
        print(e)
    return web_response
